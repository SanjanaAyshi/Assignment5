What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects, allowing scripts to manipulate the document's content, structure, and styles.

how do you select an element from DOM?

ou can select an element from the DOM in one line using document.querySelector()

what is event delegation in DOM and why is it important?

Event delegation in the DOM is a technique where a single event listener is added to a parent element to manage events for multiple child elements, improving performance and simplifying code by reducing the number of event listeners.

How can you manipulate an attribute of an element and style using DOM?

To manipulate an attribute and style of an element using the DOM, you can do it in a single line by combining the setAttribute() method for modifying attributes and directly accessing the style property to change CSS styles.
